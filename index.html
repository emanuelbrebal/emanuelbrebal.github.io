BOOTSTRAP

Bootstrap é um framework CSS que facilita a criação de sites responsivos e modernos, oferecendo componentes prontos e uma base sólida para design front-end.

PHP

PHP é uma linguagem de programação amplamente usada para o desenvolvimento de aplicações web dinâmicas. É conhecida por sua simplicidade, flexibilidade e integração com bancos de dados.

LARAVEL

Laravel é um framework PHP que simplifica o desenvolvimento web com uma abordagem elegante e ferramentas robustas, como autenticação, rotas e acesso a bancos de dados.

POSTGRESQL

PostgreSQL é um banco de dados relacional avançado e de código aberto. Reconhecido por sua confiabilidade e suporte a recursos modernos, é ideal para aplicações de alta complexidade.

Github e GIT

Git é um sistema de controle de versão que permite rastrear alterações no código e colaborar com outros desenvolvedores.
Já o GitHub é uma plataforma baseada em nuvem que hospeda repositórios Git, facilitando o compartilhamento, a revisão de código e o trabalho em equipe.

GODOT 4

Godot é um engine de desenvolvimento de jogos 2D e 3D, poderoso e de código aberto.
Ele oferece uma interface intuitiva e ferramentas flexíveis para criar experiências interativas.
